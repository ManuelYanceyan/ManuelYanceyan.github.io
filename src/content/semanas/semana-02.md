## Resumen

Durante la segunda semana se desarrollaron los fundamentos prácticos para la construcción de páginas web utilizando Lenguaje de Marcado de Hipertexto (HTML) y Lenguaje Marcado Extensible (XML). Asimismo, se abordó el diseño visual de las páginas web mediante Hojas de Estilo en Cascada (CSS3), enfatizando la correcta estructuración del contenido, el diseño responsivo y la compatibilidad entre navegadores.

![Semana 02](/imagenes/semana2.png)


## Clase

- **Construcción de páginas web con HTML y XML.**  
  Se explicó el uso de HTML como lenguaje base para estructurar páginas web y XML como lenguaje orientado al almacenamiento e intercambio de datos. Se analizaron sus diferencias, objetivos y escenarios de uso dentro del desarrollo web moderno.

- **Estándar Open Web Platform.**  
  Se introdujo el concepto de Open Web Platform como el conjunto de tecnologías abiertas que permiten crear aplicaciones web interoperables, accesibles y compatibles entre distintos navegadores y dispositivos.

- **Estructura de documentos HTML y XML.**  
  Se revisó la estructura básica de un documento HTML y XML, destacando la jerarquía de etiquetas, el correcto anidamiento de elementos y la organización lógica del contenido.

- **Etiquetas principales de HTML5.**  
  Se estudiaron las etiquetas semánticas de HTML5 y su importancia para mejorar la accesibilidad, la mantenibilidad del código y el posicionamiento en motores de búsqueda.

- **Árbol DOM y ciclo de vida de una página web.**  
  Se explicó el Document Object Model (DOM), el árbol de nodos, el proceso de renderizado del navegador y el ciclo de vida de una página web desde su carga hasta su visualización.

- **Optimización para motores de búsqueda (SEO).**  
  Se introdujeron conceptos básicos de SEO, resaltando el uso adecuado de etiquetas HTML, estructura semántica y buenas prácticas para mejorar la visibilidad en buscadores.

- **Compatibilidad entre navegadores.**  
  Se analizaron problemas comunes de compatibilidad y la importancia de seguir estándares web para garantizar una correcta visualización en distintos navegadores.

- **Diseño de páginas web con CSS.**  
  Se explicó el uso de Hojas de Estilo en Cascada para definir la presentación visual de una página web, separando estructura y contenido del diseño.

- **Estándar CSS3 y sintaxis básica.**  
  Se revisaron los selectores, reglas y propiedades de CSS3, así como su correcta aplicación sobre elementos HTML.

- **Modelos de visualización y diseño.**  
  Se estudiaron los modelos block, inline e inline-block y el uso de unidades de medida absolutas y relativas.

- **Diseño moderno con Flexbox y Grid Layout.**  
  Se explicó el uso de Flexbox y Grid Layout para crear diseños flexibles, estructurados y adaptables a distintos tamaños de pantalla.

- **Diseño web responsivo.**  
  Se revisaron los conceptos de diseño fluido y responsivo para asegurar una correcta visualización en dispositivos móviles, tabletas y computadoras.

## Laboratorio

- **Laboratorio 02A:**  
  Diseño y desarrollo de una página web aplicando Flexbox, diseño fluido y diseño web responsivo.  
  Implementación práctica utilizando HTML y CSS3, asegurando compatibilidad entre dispositivos y navegadores.
