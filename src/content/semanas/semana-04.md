## Resumen

Durante la cuarta semana se abordó el desarrollo de páginas web Front-End utilizando JavaScript y TypeScript, enfocándose en dotar de interactividad y dinamismo a las aplicaciones web. Se estudiaron los fundamentos del lenguaje JavaScript, sus tipos de datos, estructuras de control y funciones, así como la manipulación del DOM y la creación de animaciones y gráficos mediante Canvas. Estas competencias permitieron comprender cómo el navegador interpreta y ejecuta código para responder a la interacción del usuario.

## Clase

- **Introducción a JavaScript y TypeScript.**  
  Se explicó el rol de JavaScript como lenguaje principal para el desarrollo Front-End y el uso de TypeScript como una extensión que incorpora tipado estático, mejorando la calidad y mantenibilidad del código.

- **Motores JavaScript.**  
  Se analizaron los motores JavaScript utilizados por los navegadores para interpretar y ejecutar código, comprendiendo su importancia en el rendimiento y la ejecución de aplicaciones web.

- **Variables, operadores y tipos de datos.**  
  Se estudiaron los distintos tipos de datos como números, cadenas de texto, booleanos, objetos y otros tipos especiales, así como el uso de operadores aritméticos, lógicos y de comparación.

- **Estructuras de control y bucles.**  
  Se trabajaron estructuras condicionales y bucles para controlar el flujo del programa, permitiendo la toma de decisiones y la repetición de tareas dentro de una aplicación web.

- **Funciones en JavaScript.**  
  Se explicó la sintaxis e invocación de funciones, incluyendo funciones anónimas, funciones objeto, funciones flecha, funciones auto invocadas y closures, resaltando su utilidad para la reutilización de código y el manejo del alcance de variables.

- **Manipulación del DOM con JavaScript.**  
  Se abordó la interacción con el Document Object Model (DOM), utilizando métodos y propiedades para acceder, modificar y estilizar elementos HTML de manera dinámica.

- **Animaciones y gráficos con Canvas.**  
  Se introdujo el uso del elemento Canvas para la creación de animaciones y gráficos, permitiendo dibujar y controlar elementos visuales mediante JavaScript.

- **Buenas prácticas en programación Front-End.**  
  Se resaltó la importancia de escribir código claro, estructurado y reutilizable, así como el manejo adecuado de eventos y recursos para mejorar la experiencia del usuario.
