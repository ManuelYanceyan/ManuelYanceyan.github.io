## Resumen

Durante la décima semana se introdujo el lenguaje de programación Python como herramienta fundamental para el desarrollo de aplicaciones backend. Se estudiaron los conceptos básicos del lenguaje, su sintaxis, tipos de datos y estructuras de control. Asimismo, se abordaron principios de programación orientada a objetos, manejo de excepciones y modularización del código, permitiendo desarrollar aplicaciones estructuradas y mantenibles.

## Clase

- **Introducción al lenguaje Python.**  
  Se presentó Python como un lenguaje de programación interpretado, de alto nivel y multipropósito, destacando su facilidad de aprendizaje y amplio uso en el desarrollo de aplicaciones web, científicas y de automatización.

- **Sintaxis básica, indentación y comentarios.**  
  Se explicó la importancia de la indentación en Python como parte fundamental de la sintaxis, así como el uso de comentarios para documentar y mejorar la legibilidad del código.

- **Variables y tipos de datos.**  
  Se estudiaron los tipos de datos básicos como números, cadenas de texto y booleanos, así como la declaración y uso de variables dentro de un programa.

- **Estructuras de datos.**  
  Se abordó el uso de listas y tuplas para el manejo de colecciones de datos, así como diccionarios para el almacenamiento de pares clave–valor.

- **Condicionales y estructuras de control.**  
  Se explicó el uso de estructuras condicionales y bucles para controlar el flujo de ejecución del programa y la toma de decisiones.

- **Funciones imperativas y retorno de valores.**  
  Se trabajó la definición de funciones, el paso de parámetros, el retorno de valores y la reutilización de código mediante funciones.

- **API imperativa.**  
  Se introdujo el concepto de programación imperativa y el uso de funciones y métodos que modifican el estado del programa.

- **Programación Orientada a Objetos en Python.**  
  Se explicaron los principios de encapsulamiento, herencia múltiple y polimorfismo, aplicados al desarrollo de aplicaciones estructuradas.

- **Clases, atributos e instancias.**  
  Se estudió la creación de clases, definición de atributos y generación de instancias para modelar objetos del mundo real.

- **Métodos, constructores y destructores.**  
  Se explicó el uso de métodos dentro de las clases, así como la función de los constructores y destructores en la inicialización y liberación de recursos.

- **Manejo de excepciones.**  
  Se abordó el manejo de errores mediante excepciones, permitiendo controlar situaciones inesperadas durante la ejecución del programa.

- **Módulos y reutilización de código.**  
  Se explicó la creación e importación de módulos para organizar el código y facilitar su reutilización en distintos proyectos.

## Laboratorio

- **Laboratorio 10:**  
  Desarrollo de una aplicación utilizando Python, aplicando estructuras de control, funciones y programación orientada a objetos.  
  Implementación de clases, manejo de excepciones y uso de módulos como parte de una solución backend básica.
