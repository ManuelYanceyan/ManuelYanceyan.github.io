## Resumen

Durante la quinta semana se abordó el desarrollo de aplicaciones Front-End utilizando Frameworks JavaScript modernos. Se estudió la estructura y funcionamiento de estos frameworks bajo el enfoque de Client Side Rendering, así como la gestión de dependencias y el uso de herramientas para la creación de proyectos. Además, se trabajó el desarrollo basado en componentes, el uso de props y children, y distintas formas de aplicar estilos en aplicaciones Front-End.

## Clase

- **Introducción a los Frameworks JavaScript para Front-End.**  
  Se explicó el uso de frameworks JavaScript como herramientas que facilitan la creación de aplicaciones web dinámicas, escalables y mantenibles, enfocadas en la experiencia del usuario.

- **Estructura y operación de un Framework JS.**  
  Se analizó la arquitectura general de un framework Front-End, comprendiendo el flujo de ejecución y el concepto de Client Side Rendering, donde la renderización ocurre principalmente en el navegador.

- **Gestión de dependencias y creación de proyectos.**  
  Se revisaron herramientas para la creación de proyectos y la administración de dependencias, permitiendo organizar librerías, paquetes y configuraciones necesarias para el desarrollo de aplicaciones modernas.

- **Desarrollo basado en componentes.**  
  Se introdujo el concepto de componentes como unidades reutilizables de interfaz, explicando el uso de JSX para combinar lógica y estructura visual en un mismo archivo.

- **Uso de props y children.**  
  Se explicó el paso de datos entre componentes mediante props y children, permitiendo la comunicación entre componentes padre e hijo y la construcción de interfaces dinámicas.

- **Gestión de estilos en Frameworks JS.**  
  Se abordaron diferentes formas de aplicar estilos en aplicaciones Front-End, incluyendo estilos en línea, hojas de estilo tradicionales, módulos de estilos y estilos basados en componentes.

- **Integración con Frameworks CSS.**  
  Se explicó cómo integrar frameworks CSS dentro de aplicaciones JavaScript para mantener consistencia visual y facilitar el diseño responsivo.

- **Buenas prácticas en desarrollo Front-End.**  
  Se enfatizó la importancia de la reutilización de componentes, organización del código, separación de responsabilidades y mantenimiento de estilos coherentes.

## Laboratorio

  Desarrollo de una aplicación Front-End utilizando un Framework JavaScript.  
  Implementación del paso de datos mediante props y children, mostrando la información en un componente hijo y aplicando estilos adecuados para una correcta presentación visual.
